# Calculadora de IMC

Este é um **projeto pessoal** desenvolvido para praticar HTML, CSS e JavaScript puro. Trata-se de uma **calculadora de IMC (Índice de Massa Corporal)** simples, que permite calcular o IMC de forma rápida e visualizar a classificação correspondente.

## Funcionalidades

- Recebe peso (kg) e altura (m) do usuário.
- Calcula o IMC automaticamente.
- Exibe a faixa do IMC:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade grau 1, 2 ou 3
- Validações para entradas inválidas.

## Como usar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` no navegador.
3. Digite seu peso e altura e clique em **Enviar**.
4. Veja seu IMC e a classificação correspondente.

## Tecnologias usadas

- HTML5
- CSS3
- JavaScript (ES6+)

**Observação:** Projeto criado para fins de prática pessoal, **livre para modifições**.
